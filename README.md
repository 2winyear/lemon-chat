## 시작 가이드
### Requirements
For building and running the application you need:
npm, node, vite


### Installation
``` bash
$ git clone https://github.com/2winyear/lemon-chat.git
$ cd lemon-chat
```
#### Backend
```
$ cd server
$ npm install
$ npm run dev
```

#### Frontend
```
$ cd client
$ npm install vite -f 
$ npm run dev
```
